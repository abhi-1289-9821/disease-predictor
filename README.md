# 🩺 Disease Prediction System

An AI-powered web application that predicts possible diseases based on user-selected symptoms using Machine Learning.

---

## 🚀 Live Demo
👉 https://your-app-name.streamlit.app

---

## 📌 Features

- 🔍 Search and select symptoms easily  
- 🤖 Machine Learning based predictions  
- 🏆 Shows Top 3 probable diseases  
- 📊 Confidence score for each prediction  
- 🎨 Clean and professional UI (Streamlit)  

---

## 🧠 Machine Learning

- Models Used:
  - Logistic Regression ✅ (Best)
  - Random Forest
  - KNN
  - Gradient Boosting

- Evaluation:
  - Compared multiple models  
  - Selected best based on accuracy  

---

## 📊 Dataset

- Large dataset with:
  - 2,40,000+ records  
  - 300+ symptoms  
  - Multiple disease classes
